# 🌟 Project Overview

The **AI Hospital Scheduler System** is an intelligent healthcare management platform that automates:

- 🧑‍⚕️ Doctor Scheduling  
- 📅 Smart Appointment Booking  
- 🤖 AI-Driven Doctor Selection  
- ⏳ Wait Time Prediction  
- 🚑 Priority Based Scheduling  
- 🎫 Token Based Queue System  
- 💬 AI Chatbot Booking  
- ⭐ Patient Feedback & Ratings  
- 📊 Admin Analytics Dashboard  

This system simulates **real hospital workflow automation using AI-driven scheduling algorithms.**

# 🧠 AI & Algorithms Used

### 1️⃣ Heuristic Doctor Selection Algorithm
Used to select the **best doctor automatically**.

Factors considered:
- Queue length
- Doctor experience
- Consultation time
- Patient priority

Formula concept: Score = w1(queue factor) + w2(priority weight) + w3(experience)
